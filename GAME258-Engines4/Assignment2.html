Assignment 2

Make the Scene class capable of storing the Refs to all the Actors in the the the current Scene. (16/25)


The idea is that if we store the Refs to all the Actor in the Scene within the Scene so that we would have a single node (std::vector) to Render, Update, and pass events into. This will simlify every aspect of the Scene including removing all Refs to individual Actors in the Scene#.h header file. All we'll need to do now is loop over all Actors in the list Update(), HandleEvents() and Render() everyone. One way to do this is to have the Scene inherit from Actor. However, the Actor doesn't allow for adding multiple components of the same type. Use the Actor class a guide not the base class.
Next, remove all the redundant creation of the shaders, the textures and the meshes. Just share the Ref with each actor.
